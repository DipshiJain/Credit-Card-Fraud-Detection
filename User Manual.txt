To run tkinter UI file you will need to install tkinter first.
Use "pip install tk" in command prompt or to install in jupyter notebook use ! prefix before the command. To install it in ubuntu you can use "sudo apt-get install python-tk" command. 
To run this file you just need to run it and wait for few minutes for the tkinter window to get open.
